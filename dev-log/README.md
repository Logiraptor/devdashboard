# Architecture Decision Log

This directory contains records of significant architectural decisions made in the devdeploy project.

## Purpose

- Capture the context and reasoning behind major decisions
- Provide a historical record for future reference
- Help onboard new contributors by explaining "why" not just "what"

## Format

Each entry follows this template:

```markdown
# Title of Decision

**Date**: YYYY-MM-DD
**Status**: proposed | accepted | deprecated

## Context
What prompted this decision?

## Decision
What was decided?

## Consequences
What are the tradeoffs and implications?
```

## Naming Convention

Files are named: `YYYY-MM-DD-topic-name.md`
